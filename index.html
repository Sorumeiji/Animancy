Animancy Designs

Animancy creates the web for content creators and art creatives. Marketing, strategy, and branding is costly and tough to figure out alone.

That is why I am here help you to take the first step to building an online presence that tells the world about what you want to do.

The Process

01. Planning

We want to make sure that your website markets to your brand, and target audience. Helping you to plan and write the content, helps make some big design decisions before getting to any visuals.

02. Layout Drafting

We will work with you to create a mockup that represent your idea. We know the first design will not always be the best, so we consistently provide updates and receive feedback, making sure your target audience will love your design as much as you do.

03. Polish

Before launching your website we will, together, always make sure your website is ready to go and is optimized. Our support is always ready and available any time you need us, for no cost.

Other Services.

Data Intergration

If you want to display information, redeems, top cheer, etc. We can intergrate that data using various API's such as Twitch, Streamlabs, Muxy, and StreamElements.

Broadcast Graphics and Controls

We use NodeCG to create broadcast graphics. This services allows one to help run complicated parts of thier stream smoothly, and create new ideas using the power of node.js and the browser.

Web Animation OBS

If you need to show a cool effect, or data during your stream, we can help. We create code that runs in the Browser CLR tool in OBS that allows you to showcase various concepts and information to viewers.
